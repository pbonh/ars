# Developer Tools

This repo supports installing terminal developer tools & packages via homebrew (default), mise,
devbox, and cargo (Non-Root).

Base provider examples only require `tool_provider`. Git identity and GitHub username are optional
and can be applied separately with the `dev` role (`dev.yml`) when desired.

The options are presented in the recommended order:

1. Homebrew (default)
2. Mise
3. Devbox
4. Non-Root
