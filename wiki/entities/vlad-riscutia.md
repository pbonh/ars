## Overview

Vlad Riscutia is a software engineer and the author of *Programming with Types* (Manning Publications). His professional background spans modern C++ and functional programming, with particular interest in generic programming, type systems, and software correctness.

## Characteristics

- Deeply influenced by Alexander Stepanov’s work on generic programming (C++) and by Haskell’s advanced type system.
- Advocates for bridging theoretical type system research with practical, day-to-day software engineering.
- Uses TypeScript as a teaching vehicle because its syntax is accessible while its type system is powerful enough to demonstrate functors, monads, and higher-kinded types.

## Common Strategies

- Distills dense theoretical material (category theory, type theory) into informal, example-driven explanations.
- Emphasizes compile-time guarantees over run-time testing for eliminating whole classes of bugs.
- Recommends wrapping primitive types into domain-specific types to prevent misinterpretation.
