## Overview

Earendil Works is the organization that develops and maintains Pi, a minimal terminal coding agent harness. The project is hosted on GitHub under the `earendil-works` organization.

## Characteristics

- **Primary product**: Pi coding agent (`@earendil-works/pi-coding-agent`)
- **Ecosystem**: Includes `pi-tui`, `pi-ai`, and related monorepo packages
- **Distribution**: npm registry and curl-based installer (`https://pi.dev/install.sh`)
- **Repository**: https://github.com/earendil-works/pi

## Common Strategies

- Install Pi via npm or the official installer script
- Follow the monorepo structure in `pi-mono` for SDK and extension development
- Use the GitHub repository for issue tracking and feature requests
