## Overview

Anthropic is an AI research and safety company best known for developing the Claude family of large language models. In the context of the Agent Skills ecosystem, Anthropic is the original creator of the Agent Skills open standard, which it released to enable portable, reusable skill documents across the growing landscape of AI agents.

## Characteristics

- **Primary products**: Claude ( conversational AI ), Claude Code ( agentic coding tool ), and the Claude API platform.
- **Standards work**: Authored and open-sourced the Agent Skills format, now maintained by the broader agentskills.io community.
- **Research focus**: AI safety, alignment, and building reliable, steerable systems.
- **Business model**: API subscriptions and enterprise partnerships.

## Common Strategies

- Use Claude Code with custom Agent Skills to standardize code-review or refactoring procedures across a team.
- Publish internal workflows as Agent Skills so they can be reused across Anthropic-compatible agents ( Claude Code, Cursor, VS Code Copilot, etc. ).
- Contribute improvements to the Agent Skills specification via the public GitHub repository.
