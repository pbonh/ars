## Overview

**Marp** (Markdown Presentation Ecosystem) is a markdown-based slide deck format. It allows users to write presentations in plain markdown with special directives, then export to HTML, PDF, or PowerPoint. Obsidian has a plugin for Marp, making it a natural fit for generating presentations directly from LLM wiki content.

## Characteristics

- **Format:** Markdown with YAML frontmatter and CSS-based theming.
- **Export targets:** HTML, PDF, PPTX, and live preview.
- **Obsidian integration:** Native plugin available in the Obsidian community plugins catalog.
- **Authoring model:** Write slides as markdown sections; no GUI required.

## Common Strategies

- **Presentation from wiki:** Ask the LLM to synthesize a comparison or analysis from wiki pages, then output it as a Marp deck for sharing or presenting.
- **Quick pitch decks:** Generate investor or stakeholder updates from accumulated project wiki pages.
- **Documentation talks:** Convert a deep concept page or synthesis into a slide deck for a team walkthrough.
