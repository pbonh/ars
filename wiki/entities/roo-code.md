## Overview

Roo Code is an open-source AI coding agent that embeds an entire dev team inside your editor. It emphasizes deep project-wide context, multi-step agentic coding, and developer-centric flexibility. Roo Code supports Agent Skills, enabling users to add custom capabilities via the standard skill format.

## Characteristics

- **Type**: Open-source IDE extension / coding agent.
- **Philosophy**: Outpaces closed tools by giving developers control over context and behavior.
- **Skills support**: Loads Agent Skills according to the agentskills.io specification.
- **Project context**: Maintains awareness across the whole codebase for accurate, large-scale changes.

## Common Strategies

- Install Roo Code skills from the community or write private ones for internal frameworks.
- Use Roo Code's multi-step agent mode with a skill to perform large refactors that touch many files consistently.
- Contribute improvements to Roo Code's skill-handling logic back to the open-source repository.
