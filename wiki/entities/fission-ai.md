## Overview

Fission AI is the organization that created and maintains [[entities/openspec]], an open-source specification and planning framework for AI-assisted software development. The company distributes OpenSpec via npm under the `@fission-ai` scope and engages with the community through GitHub issues and Discord.

## Characteristics

| Attribute | Detail |
|-----------|--------|
| Product | OpenSpec — specification framework for AI coding assistants |
| Distribution | npm (`@fission-ai/openspec`), Nix, GitHub source |
| Community | Discord (discord.gg/YctCnvvshC) |
| Issue tracking | https://github.com/Fission-AI/OpenSpec/issues |

## Common Strategies

- **Open-source core:** OpenSpec is developed in the open on GitHub, with community schema contributions accepted (e.g., `superpowers-bridge` community schema).
- **Broad AI-tool compatibility:** Fission AI prioritizes supporting a wide range of AI coding assistants rather than tying OpenSpec to a single vendor.
- **Iterative public development:** The OPSX workflow and workspace features were released as rough-but-usable and refined based on community feedback.
