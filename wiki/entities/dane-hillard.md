## Overview

Dane Hillard is a lead web application developer at ITHAKA, a nonprofit in higher education. He is the author of *Practices of the Python Pro* (Manning, 2020), a book that teaches software design principles to Python developers ready to move beyond scripting into building complex, maintainable systems.

## Characteristics

- Background in building inference engines for telemetry data and ETL pipelines for bioinformatics applications.
- First forays into programming included custom styling for MySpace, scripting for Rhinoceros 3D, and modding the MS-DOS game Liero.
- Enjoys creative coding and seeks ways to combine software with music, photography, and food.
- Active speaker at Python and Django conferences internationally.

## Common Strategies

- Teaches software design through iterative, hands-on exercises rather than pure theory.
- Uses a progressively built sample application (Bark) to demonstrate separation of concerns, extensibility, and loose coupling in a real codebase.
- Advocates for "make it work, make it right, make it fast" as a cyclical, incremental practice.
