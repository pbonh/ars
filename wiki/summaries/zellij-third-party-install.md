## Overview

Zellij is packaged by third-party maintainers for several Linux distributions and macOS package managers. These packages are not officially affiliated with the Zellij maintainers but are documented for user convenience.

## Key Claims

- Official Arch Linux extra repository and AUR both carry Zellij packages.
- Fedora packages are available via a COPR repository.
- macOS supports both Homebrew and MacPorts.
- Void Linux includes Zellij in its official repositories.
